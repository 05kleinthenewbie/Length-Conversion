# 📏 Length Converter

A simple web app to convert between **centimeters, inches, and feet**.  
Built with **HTML, CSS, and JavaScript**.

---

## 🚀 Features
- Convert:
  - Inches ➝ Centimeters
  - Centimeters ➝ Inches
  - Centimeters ➝ Feet + Inches
- Clean and responsive design.
- Instant conversion with a single click.
